# Шпаргалка по оформлению презентации — РТУ МИРЭА

Выжимка из брендбука (266 стр.), официального шаблона презентации и шаблона защиты КБ-9. Только то, что нужно для слайдов к защите ВКР. Сверено с примером отличницы (Иванова).

---

## 0. Главное правило

- Базовый файл — **`Шаблон-презентации-к-защите_КБ9.pptx`**. В нём задан перечень обязательных слайдов для защиты ВКР на кафедре КБ-9, профиль «Технологии визуального программирования».
- Брендбук и шаблоны утверждены в феврале 2026. Источник: https://www.mirea.ru/mediapage/brend-platforma-rtu-mirea/
- Внутри слайдов пространство организуешь как угодно, **но правила брендбука нарушать нельзя**.
- Размеры шрифтов менять можно. **Подзаголовки слайдов — обязательны** (их надо переписать под свою тему).
- Формат слайда — **16:9, 960 × 540 pt** (как в шаблоне и у Ивановой).

---

## 1. Палитра (точные цвета)

Какую брать — зависит от того, чей это слайд. Шаблон КБ-9 на титуле использует **бирюзовую палитру Института кибербезопасности**, остальное держится на общей сине-бирюзовой гамме университета.

### Общая палитра МИРЭА (основная, для большинства слайдов)
| Роль | HEX | RGB |
|---|---|---|
| Тёмно-синий (основной) | `#355CA8` | 53, 92, 168 |
| Бирюзово-синий (доп.) | `#5FC1CF` | 95, 193, 207 |
| Голубой (доп.) | `#A7D9DE` | 167, 217, 222 |
| Светло-серый (фон/доп.) | `#E6E6E6` | 230, 230, 230 |
| Градиент | `#355CA8` → `#3772B3` → `#5FC1CF` | — |

### Палитра Института кибербезопасности и цифровых технологий (акценты КБ-9)
| Роль | HEX | RGB |
|---|---|---|
| Основной (бирюза) | `#006A81` | 0, 106, 129 |
| Тёмный доп. | `#004352` | 0, 67, 82 |
| Светлый доп. | `#94CCD9` | 148, 204, 217 |
| Светло-серый | `#E6E6E6` | 230, 230, 230 |

> Фон слайдов — белый. Текст — тёмно-синий/бирюзовый. Цвета вне палитры — нельзя.

## 2. Шрифты

- **Основной — Helvetica Neue.** Рабочие начертания: **Roman, Medium, Bold** (ими набрано большинство макетов).
- **Запасной — Arial** (Regular, Bold) — если Helvetica Neue недоступна или для текстовой части документов. Для этого случая есть готовый вариант шаблона `..._arial.pptx`.
- Практика: **заголовки** — Bold, тёмно-синий; **подзаголовки и текст** — Roman/Regular.

> Если на твоём ПК нет Helvetica Neue — бери Arial-вариант шаблона, не подменяй на сторонний похожий шрифт.

---

## 3. Логотип и герб

**Для презентации используется ЛОГОТИП (круг «РТУ МИРЭА»), а не герб.** Герб — только для VIP-продукции и торжественных мероприятий госуровня, на обычной презентации он неуместен.

Версии логотипа по фону:
- цветной — на белом однородном фоне;
- белый — на корпоративном синем фоне;
- синий — на бирюзово-синем фоне.

Размещение: **логотип в правом верхнем углу** — крупно на титуле, мелко на контентных слайдах (как в официальном шаблоне).

**Охранное поле** (пустое пространство вокруг логотипа/герба) = **2/5 от ширины знака** со всех сторон. Внутрь ничего не залезает.

Запрещено:
- менять цвета, градиент, пропорции или шрифт логотипа;
- ставить логотип на неоднородный фон / фото;
- использовать одновременно логотип университета и герб;
- использовать одновременно основной логотип и логотип с названием института.

## 4. Композиция слайда (по официальному шаблону)

**Титул:**
- белый фон;
- вертикальная **градиентная полоса слева** (бирюза → синий);
- стилеобразующий элемент — **большой круг с градиентным «пятном»**;
- заголовок — тёмно-синий Bold, слева; ниже дата;
- логотип-круг — в правом верхнем углу;
- слоган **«ВМЕСТЕ СОЗДАЁМ БУДУЩЕЕ»** внизу, капителью, с разрядкой.

**Контентный слайд:**
- заголовок — тёмно-синий Bold, слева сверху; мелкий логотип справа сверху;
- слева — **акцент-панель** (синяя скруглённая плашка с градиентом) с ключевым тезисом белым;
- контент — в **карточках со скруглёнными углами и бирюзовым контуром**;
- иконки — линейные, бирюзовые;
- **номер слайда** — внизу справа.

## 5. Структура (ориентир — пример Ивановой, 14 слайдов)

1. **Титул** — тема ВКР, ФИО, руководитель, кафедра, направление/профиль.
2. **Введение** — цель, задачи, актуальность, новизна, практическая значимость (на одном слайде, блоками).
3–4. **Аналитический раздел** — анализ существующих решений, анализ потребностей ЦА, выбор технологий.
5. **Постановка задачи** — функциональные требования, структура приложения.
6. **Функциональная схема** приложения.
7. **6.1 Информационное обеспечение** — БД, подключение.
8–10. **6.2–6.4 Программное обеспечение** — серверная часть, интеграции, клиентская часть.
11. **6.5 Техническое обеспечение** — развёртывание.
12. **План дальнейшего развития.**
13. **Заключение** — цель достигнута, задачи выполнены.

> Логика «три обеспечения» (информационное → программное → техническое) — ровно то, что в твоём разделе 3. У Ивановой каждое обеспечение = свой блок слайдов с подзаголовком «6.x …».
>
> Шаблон КБ-9 задаёт **12 обязательных слайдов** (первый — технический, с инструкцией, его удаляешь). Подзаголовки в нём — обязательны, перепиши под свою тему.

## 6. Чек-лист перед сдачей

- [ ] Формат 16:9, белый фон, шрифт Helvetica Neue (или Arial-вариант).
- [ ] Заголовки — Bold тёмно-синий; подзаголовки переписаны под свою тему.
- [ ] Цвета только из палитры (синяя МИРЭА + бирюза КБ для акцентов).
- [ ] Логотип в правом верхнем углу, охранное поле 2/5, версия под фон.
- [ ] Технический слайд-инструкция из шаблона удалён.
- [ ] Номера слайдов проставлены.
- [ ] Все «жёлтые» места из шаблона заменены своим содержимым.
