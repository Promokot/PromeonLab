# Design-brief для Claude Design (вставить промптом)

> Это инструкция для генерации презентации в режиме слайд-деков. Вставь этот текст промптом,
> а файлы из пака (`01_deck-content.md`, `assets/`) приложи к сообщению.

---

Ты — дизайнер презентаций. Сделай слайд-дек к защите ВКР по фирменному стилю РТУ МИРЭА
(Институт кибербезопасности, КБ-9). Контент каждого слайда — в приложённом файле
`01_deck-content.md`. Держись бренда строго, но композицию внутри слайда можешь решать сам.
Финал — экспорт в редактируемый **.pptx**.

> **ИСТОЧНИК СТИЛЯ — приложённая презентация Ивановой (`пример Иванвоа no vid.pptx`).**
> Это эталон оформления в нужной палитре (Институт кибербезопасности). **Копируй из неё стиль,
> а НЕ содержание** — тема у неё другая. Оттуда же бери настоящий **логотип** (кольцо «РТУ МИРЭА»)
> и **декоративные сферы** (бирюзово-синие). Дополнительный быстрый ориентир — склейка
> `ref_ivanova_examples_collage.png`.

## Общие параметры
- Формат **16:9**, всего **14 слайдов** (по числу секций в `01_deck-content.md`, разделены `___`).
- Фон слайдов — **белый**. Много «воздуха», не перегружай.
- Язык — русский.

## Палитра (только эти цвета, см. `assets/palette_swatch.png`)
- Основной синий **#355CA8** — заголовки, плашки, акценты.
- Бирюза ИКБ **#006A81** — второй акцент: ключевые слова, контуры карточек, иконки, стрелки.
- Доп.: синий **#3772B3**, бирюза **#5FC1CF**, светлая бирюза **#94CCD9**, серый фон **#E6E6E6**.
- Градиент для декора: **#355CA8 → #3772B3 → #5FC1CF**.
- Текст — тёмно-синий (#355CA8 / почти чёрно-синий), НЕ чистый чёрный. Цвета вне палитры нельзя.

## Шрифт
- **Helvetica Neue** (начертания Roman/Medium/Bold); если недоступен — **Arial**.
- Заголовки — Bold тёмно-синий; подзаголовки и текст — Regular.

## Логотип
- Бери **настоящий логотип РТУ МИРЭА из приложённой .pptx Ивановой** (кольцо с надписью «РТУ МИРЭА»),
  НЕ герб и НЕ выдумывай свой знак.
- Размещение — **правый верхний угол**: крупно на титуле, мелко на контентных слайдах.
- Охранное поле вокруг знака = 2/5 его ширины; не растягивать, не менять пропорции/цвет.

## Декор
- Фирменные градиентные «сферы» — **бирюзово-синие, из .pptx Ивановой** (см. её титул и склейку).
  Бледно, по краю/углу титула и заключения, как фон. Не лепить на каждый слайд. Оранжевых/красных
  сфер из общего шаблона НЕ использовать — это палитра другого института.

## Два мастер-макета (ориентиры — приложённая .pptx Ивановой и склейка `ref_ivanova_examples_collage.png`)

**Титул (слайд 1):**
- Слева сверху мелким тёмно-синим: институт → кафедра → направление → профиль.
- Логотип-круг — справа сверху, крупно.
- По центру-справа — крупные бледные градиентные «сферы».
- «ВКР на тему:» + жирный тёмно-синий заголовок темы — слева по центру.
- «Выполнил / Руководитель» — внизу слева.
- Внизу слоган «ВМЕСТЕ СОЗДАЁМ БУДУЩЕЕ» — капитель, вразрядку, светлый.

**Контентный слайд (2–14):**
- Заголовок Bold тёмно-синий — слева сверху; под ним серым/бирюзовым — строка-подзаголовок.
- Мелкий логотип — справа сверху. Номер слайда — внизу справа.
- Если есть картинка/схема: **левые ~60% — изображение, правые ~40% — текст** (как в референсе).
- Текст — короткие тезисы; ключевые термины — бирюзой Bold; нумерованные списки — компактные.
- Контент-блоки — карточки со скруглёнными углами и тонким бирюзовым контуром.
- Иконки — линейные, бирюзовые.
