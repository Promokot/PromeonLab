# Пак для Claude Design — презентация ВКР PromeonLab

## Как собрать презентацию (порядок)
1. Открой Claude (веб) → режим слайд-деков (Design).
2. Перетащи в чат **всё из папки `_upload-these/`** (12 файлов) **+ саму презентацию Ивановой**
   `пример Иванвоа no vid.pptx` (она — эталон стиля; лежит в `docs/vkr/inbox/`).
   Итого ≤ 13 вложений — под лимит 20.
3. Текст из `00_design-brief.md` вставь **как сообщение** (это промпт-инструкция).
4. Сгенерируй дек → проверь → экспортируй в `.pptx`.

> **Важно про бренд:** логотип, сферы, точные цвета и шрифт Claude берёт из приложённой
> **.pptx Ивановой** (правильная палитра ИКБ). Склейка `ref_ivanova_examples_collage.png` —
> быстрый визуальный ориентир. Свой логотип/сферы НЕ выдумывать.
