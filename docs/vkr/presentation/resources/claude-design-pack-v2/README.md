# Пак v2 для Claude Design — точечные правки деки PromeonLab

Дека уже собрана (17 слайдов) — пак вносит в неё точечные правки, а не строит заново.

## Как скормить (порядок)

1. Открой Claude → режим слайд-деков (Design).
2. Приложи к сообщению:
   - **`база_дека.html`** — актуальная дека (основа, 17 слайдов),
   - `план_реструктуризации.md` — полная спецификация по слайдам,
   - всё из `assets/`.
3. Текст из **`ИНСТРУКЦИЯ_claude_design.md`** вставь сообщением (это промпт; в нём 4 правки).
4. Получи обновлённый HTML → проверь → при нужде попроси правки.

## Что нового в этой итерации (4 правки)
- **A. Чипы «Задача N»** на слайдах 3–4 и 6–14 (соответствие шаблону КБ-9; карта — раздел 2А плана).
- **B. Слайд 10** «Сцена/объекты/риги» → три колонки + визуал `diagram_asset_builders.svg`.
- **C. Слайд 12** «Экспорт» → структура архива (`export_bundle_tree.txt` + `export_scene_flat.json`)
  в стиле слайда хранения.
- **D. Слайд 15** «План развития» → лесенка из 4 этапов (`diagram_roadmap_staircase.svg`).

## Важно про картинки
`база_дека.html` ссылается на свою папку **`images/`**. Новые файлы из `assets/`
(`diagram_asset_builders.svg`, `diagram_roadmap_staircase.svg`, `export_bundle_tree.txt`,
`export_scene_flat.json`, `diagram_functional_v2.svg`, `scene_example_large.json`, рисунки 3.7/3.10/3.13/3.15
и логотипы стека) положи в ту же `images/`.

## Логотипы стека (слайд 4)
`stack-logos/СКАЧАТЬ_логотипы.md` — прямые ссылки на 8 логотипов. Скачать вручную, положить в `images/`.
Без них — карточки-заглушки.
