# Пак v2 для Claude Design — реструктуризация деки PromeonLab (14 → 17 слайдов)

Этот пак НЕ генерирует деку с нуля — он перестраивает уже готовую `база_дека.html`.

## Как скормить (порядок)

1. Открой Claude → режим слайд-деков (Design).
2. Приложи к сообщению:
   - `база_дека.html` — текущая дека (основа),
   - `план_реструктуризации.md` — что менять по слайдам,
   - всё из `assets/` (схемы, рисунки ВКР, JSON-примеры, палитра, референс стиля).
3. Текст из **`ИНСТРУКЦИЯ_claude_design.md`** вставь **сообщением** (это промпт).
4. Получи обновлённый HTML → проверь → при нужде попроси правки.

## Важно про картинки

`база_дека.html` ссылается на свою папку **`images/`** (бренд-сферы, логотип, уже вставленные
рисунки). Держи её рядом с HTML. Новые файлы из этого пака —
`diagram_functional_v2.svg`, `scene_example_large.json`, `animation_example.json`,
`fig_3.7/3.10/3.13/3.15` и логотипы стека — **положи в ту же `images/`**, чтобы дека их видела.

## Логотипы стека (слайд 4)

В `stack-logos/СКАЧАТЬ_логотипы.md` — прямые ссылки на 8 логотипов (Unity, C#, OpenXR, glTF,
Blender, Git, GitHub, Visual Studio). Скачать вручную (бинарники не кладу автоматически),
положить в `images/`. Без них Claude Design оставит карточки-заглушки.
