# Логотипы стека для слайда 4 — что скачать

Положи скачанные файлы рядом, в папку `stack-logos/`, под указанными именами. Потом перенесёшь их
в папку `images/` деки. Если что-то не скачать — Claude Design оставит аккуратную карточку-заглушку
с названием технологии.

Порядок карточек на слайде (сверху вниз): Unity + C#, OpenXR, glTF, Blender, Git · GitHub, Visual Studio.
