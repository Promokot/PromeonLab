# Конспект: «Требования к оформлению ВКР» (PDF, 22 стр.)

Источник: официальная методичка кафедры (Word 2019, февраль 2026); сам PDF лежит в docs/vkr/inbox/Требования к оформлению ВКР.pdf. Примеры оформления источников – Приложение Г (стр. 22 PDF). Здесь — выжимка того, что влияет на нашу работу в .md и на перенос в Word.

## Структура ПЗ
Титул → задание → СОДЕРЖАНИЕ → ВВЕДЕНИЕ → основная часть → ЗАКЛЮЧЕНИЕ → СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ → ПРИЛОЖЕНИЯ.
Нумеруются только содержательные разделы (исследовательский=аналитический, специальный, технологический, БЖД); ВВЕДЕНИЕ/ЗАКЛЮЧЕНИЕ/СОДЕРЖАНИЕ/СПИСОК — без номеров.

## Текст (для Word; в .md соблюдаем по смыслу)
- TNR 14, полуторный, выравнивание по ширине, первая строка 1,25 см; поля: лево 30, право 10, верх/низ 20 мм.
- Заголовки: ур.1 — 18 пт, ВСЕ ПРОПИСНЫЕ, с новой страницы; ур.2 — 16 пт; ур.3+ — 14 пт; все полужирные, по левому краю, отступ 1,25.
- Запрет: переносы слов, сноски внизу страницы, пустое пространство на листах, сокращения без расшифровки.
- Кавычки русские «…» (в англ. тексте "…").

## Тире/дефис/минус (ВАЖНО — анти-ИИ тоже)
- Дефис «-» — сложные слова, диапазоны (100-150 см). Без пробелов.
- Минус «–» (en dash) — только как знак вычитания.
- Тире «—» (em dash) — с пробелами, по правилам пунктуации. ⚠ Макс: длинное тире повышает ИИ-детект → в REQUIREMENTS решить, чем заменяем (перестройка фразы).
- Математические знаки (+, =, >, <) в тексте — только словами; в формулах — знаками.

## Числа
- Только арабские (кроме кварталов/полугодий — римские), без падежных окончаний; единица измерения один раз после последнего числа ряда.

## Рисунки
- После абзаца первого упоминания (или на след. странице), по центру.
- Подпись ПОД рисунком: «Рисунок 2.7 — Название», полужирный TNR 12, по центру, без точки в конце. Слово «Рисунок» не сокращать.
- Нумерация в пределах раздела: <раздел>.<порядок>.
- Ссылки: «в соответствии с Рисунком 1.2» или «(Рисунок 1.2)» — с большой буквы.

## Таблицы
- Название НАД таблицей: «Таблица 1.7 — Исходные данные» (курсив TNR 12, по левому краю, без точки).
- Перенос: «Продолжение Таблицы 8.1», нижняя черта первой части не проводится.
- Содержимое TNR 12 одинарный. Графа «№ п/п» запрещена. Диагональное деление ячеек запрещено.

## Листинги
- Надпись ПЕРЕД листингом: «Листинг 1.7 — Название» (курсив TNR 12, без точки).
- Содержимое: Courier New 10, одинарный, в рамке.
- Ссылки на строки: нумерация с шагом 10, «Листинг 3.2 {30, 50, 140}».
- Фрагменты — в тексте; ПОЛНЫЙ код программы — в приложении.

## Формулы
- Equation/MathType, по центру, пустая строка до и после; номер в круглых скобках по правому краю, в пределах раздела; нумеруются только те, на которые есть ссылки. Пояснения — «где» без двоеточия, без отступа, символы один под другим.

## Списки (маркированные/нумерованные)
- Один знак маркера на ВСЮ работу; маркер на 1,25, текст на 2,25 см.
- Маркированный: со строчной буквы, «;» в конце пункта, «.» в последнем.
- Нумерованный: с прописной, «.» в конце каждого.

## Источники
- ≥ 40 источников, год издания не ранее 2021 (⚠ в примере методички сами источники старше — но требование: 2021+).
- Список разбит ПО РАЗДЕЛАМ работы; внутри раздела — в порядке упоминания.
- Ссылки в тексте: [1.5], [2.18], с указанием страниц [2.18, c.21-25]. Точка ПОСЛЕ ссылки.
- Нумерация источников: <номер раздела>.<порядок> (например 1.1–1.x для аналитического).
- Города: М., Л., СПб.; нет пробелов в «М.:» и в инициалах.

## Приложения
- Обозначение: русские заглавные буквы с А (искл. Ё З Й О Ч Ь Ы Ъ); каждое с новой страницы.
- Первый лист раздела — перечень приложений.
- ОБЯЗАТЕЛЬНО выносятся: графический материал (Приложение А, альбомная) и код программы (стиль листинга, в рамке).
- Рисунки/таблицы в приложениях: «Рисунок Б.5».
- На все приложения должны быть ссылки в тексте.
